# 製品仕様書: hoshiyomi（星詠み）

## ビジョン
hoshiyomi は、AI とスピリチュアルを融合させた「毎日のお守り体験」を届けるブランドです。四柱推命・星読み・タロットを核に、顔出しせず・自動化された仕組みで「今日の星詠み」を SNS から LINE/Web アプリへ流し、ファンを段階的にコミュニティ・高単価セッションへ育てます。「占ってもらう」ではなく「自分の星と対話する」体験を、コーディング × ライティング × デザイン × AI で量産します。

## 目標
- **Phase 1（1〜3ヶ月）**: 月5万円達成（PDF レポート + LINE 月額 + AI Web アプリ）
- **Phase 2（4〜6ヶ月）**: 月10〜15万円（コミュニティ追加）
- **Phase 3（7〜12ヶ月）**: 月20万円+（高単価ハイブリッド鑑定追加）

## ターゲットユーザー
- **プライマリー**: 20〜40代女性、占い・星読み・スピリチュアルが好き、毎日の「お守り」として運勢を確認したい、SNS（X / Instagram / Threads）で情報収集する層
- **セカンダリー**: 自己理解・自己分析が好きな男女、四柱推命や西洋占星術の「論理」に知的興味がある層

## コアユーザージャーニー
1. X / Threads / Instagram で「今日の星詠み」投稿を見て興味を持つ
2. プロフィールリンクから hoshiyomi の LP へ遷移
3. 無料のワンポイント AI 鑑定を体験し、結果を SNS シェア
4. LINE 公式アカウントに登録し、毎朝の星詠みを受け取る
5. PDF 個人レポート（1,980円）を購入、または月額メンバーシップ（500〜980円）に登録
6. SNS と LINE で世界観に触れ続け、コミュニティ参加 → 高単価セッションへステップアップ

## 機能リスト

### スプリント 1: AI ワンポイント鑑定 LP（コア体験）
- **F01** 無料ワンポイント AI 鑑定: 生年月日を入力すると、AI が「今日のあなたへのメッセージ」を即時生成
- **F02** シェア用メッセージカード生成: 鑑定結果を SNS シェア可能な画像として保存できる

### スプリント 2: メール/LINE 集客動線
- **F03** LINE 公式アカウント登録動線: 鑑定後に LINE 友達追加で「毎朝の星詠み」が届くと案内、ワンタップで友達追加
- **F04** メールアドレス取得（ニュースレター）: 鑑定後にメール登録すれば、週1の星詠み記事が届く

### スプリント 3: 有料 PDF レポート販売
- **F05** PDF レポート商品ページ: 1,980円の詳細レポート（四柱推命 or 星読み）の内容・サンプルを掲載
- **F06** AI 個人レポート自動生成: 生年月日・出生時刻・出生地からカスタマイズ PDF を5分以内に生成
- **F07** 決済とダウンロード提供: クレカ / Apple Pay 等で決済し、購入完了後すぐに DL リンクが届く

### スプリント 4: SNS 自動投稿基盤
- **F08** 「今日の星詠み」自動投稿: X / Threads / Instagram に毎朝 同時投稿（テキスト＋画像）
- **F09** 新月・満月 特別投稿: 月相に合わせた特別メッセージを自動配信
- **F10** 投稿テンプレートの世界観統一: フォント / 色 / レイアウトがブランドガイドに沿って統一される

### スプリント 5: LINE bot プレミアム月額
- **F11** 毎朝の星詠み配信: LINE 友達全員に朝7時に無料配信
- **F12** プレミアム月額メンバーシップ: 月額500〜980円で「週運・月運・年運の詳細鑑定」が受けられる
- **F13** プレミアム会員向け追加機能: いつでもチャットで AI 鑑定リクエスト、解約は1タップ

### スプリント 6: ブランドサイト/メルマガ
- **F14** hoshiyomi ブランドサイト: ビジョン・サービス一覧・運営者の世界観を伝える LP
- **F15** メルマガ自動配信: 週1で「スピリチュアル × 自己理解」をテーマにした長文記事を配信

### スプリント 7: コミュニティ立ち上げ（Phase 2）
- **F16** 月額制クローズドコミュニティ: 月額1,500円で Discord / LINE オープンチャットに参加
- **F17** 月2回 オンラインワーク: 新月・満月にメンバー限定 Zoom ワークを開催

### スプリント 8: 高単価ハイブリッド鑑定（Phase 3）
- **F18** AI × 人間 ハイブリッド鑑定予約: 1万円/回の60分セッションを予約できる
- **F19** リピート割引 / 紹介プログラム: 既存ファンの LTV を最大化

## 受け入れ基準

### F01 — 無料ワンポイント AI 鑑定
- [ ] 生年月日入力フォームから、3秒以内に鑑定結果が表示される
- [ ] 鑑定結果は100〜400文字のメッセージ形式である
- [ ] 不正入力時は分かりやすいエラーメッセージが表示される
- [ ] スマートフォン縦表示でも文字が読みやすく折り返される
- [ ] 1日に同一ユーザーが何度でも鑑定できる（毎日変わる結果）

### F02 — シェア用メッセージカード生成
- [ ] 鑑定結果が画像（PNG）として端末に保存できる
- [ ] X / Threads / Instagram それぞれの推奨サイズで生成される
- [ ] 画像に hoshiyomi のロゴ / URL が控えめに配置される
- [ ] 「シェアする」ボタンから各 SNS にワンタップ投稿できる

### F03 — LINE 公式アカウント登録動線
- [ ] 鑑定結果ページに「毎朝の星詠みを LINE で受け取る」ボタンが表示される
- [ ] ワンタップで LINE 公式アカウント追加画面に遷移
- [ ] 友達追加後、自動応答メッセージが10秒以内に届く

### F04 — メールアドレス取得
- [ ] メール入力フォームから登録すると、即座に確認メールが届く
- [ ] 登録メアドは管理ダッシュボードで確認・エクスポートできる
- [ ] 配信解除リンクが全メールに含まれる（特商法対応）

### F05 — PDF レポート商品ページ
- [ ] 価格・内容・所要時間・サンプル抜粋が明記されている
- [ ] 「購入する」ボタンから5秒以内に決済画面に遷移する
- [ ] お客様の声（ダミー可、Phase 進行で実体化）が3件以上掲載されている

### F06 — AI 個人レポート自動生成
- [ ] 生年月日・出生時刻・出生地から個人化された PDF が生成される
- [ ] 10〜30ページの構成で、目次・章立てが整っている
- [ ] 生成時間は5分以内
- [ ] ブランドフォント / カラーが全ページで統一されている
- [ ] PDF 内に hoshiyomi のロゴ・URL が含まれる

### F07 — 決済とダウンロード提供
- [ ] クレジットカード / Apple Pay / Google Pay に対応
- [ ] 購入確認メールが10秒以内に届く
- [ ] DL リンクは購入後7日間有効
- [ ] 返金/問い合わせ動線が明記されている

### F08 — 「今日の星詠み」自動投稿
- [ ] 毎日 朝7時（JST）に自動投稿される
- [ ] X / Threads / Instagram の3媒体に同時投稿される
- [ ] 投稿失敗時は管理者にメール/Slack で通知される
- [ ] 過去30日分の投稿履歴がダッシュボードで確認できる

### F09 — 新月・満月 特別投稿
- [ ] 新月/満月の前日 21時に予告投稿される
- [ ] 当日朝7時に特別メッセージが投稿される
- [ ] 月相カレンダーが LP で閲覧できる

### F10 — 投稿テンプレートの世界観統一
- [ ] 全投稿画像に共通フォント / カラーパレットが適用されている
- [ ] ブランドガイドが `docs/brand.md` などに明文化されている
- [ ] 新規テンプレ追加時、既存と整合性チェックができる

### F11 — 毎朝の星詠み配信（LINE 無料）
- [ ] LINE 友達追加後、翌朝から自動配信される
- [ ] 朝7時±5分以内に届く
- [ ] ユーザーは配信時刻を設定変更できる

### F12 — プレミアム月額メンバーシップ
- [ ] 月額500〜980円で登録できる（価格は A/B テスト可能）
- [ ] 解約は管理画面から1タップで完了する
- [ ] 月額限定の週運・月運・年運コンテンツが配信される
- [ ] 解約後も当月末まではコンテンツが利用できる

### F13 — プレミアム会員向け追加機能
- [ ] LINE 上で「鑑定リクエスト」のリッチメニューが表示される
- [ ] リクエスト送信後30秒以内に AI 鑑定が返信される
- [ ] 1日あたりの利用回数制限が明示される

### F14 — ブランドサイト
- [ ] ファーストビューにビジョン・キャッチコピーが配置される
- [ ] サービス一覧（無料鑑定 / PDF / LINE / コミュニティ）への動線がある
- [ ] スマホ表示でレイアウトが崩れない

### F15 — メルマガ自動配信
- [ ] 週1の配信スケジュールで自動化されている
- [ ] 配信内容は管理画面でストック・予約可能
- [ ] 開封率/クリック率がトラッキングできる

### F16 — 月額制クローズドコミュニティ
- [ ] 月額1,500円で参加できる
- [ ] Discord または LINE オープンチャットに自動招待される
- [ ] 月1回以上、運営からの限定コンテンツが配信される

### F17 — 月2回 オンラインワーク
- [ ] 新月・満月の日に Zoom で開催される
- [ ] 録画が翌日までにメンバーに共有される
- [ ] アンケート/フィードバック収集の仕組みがある

### F18 — AI × 人間 ハイブリッド鑑定
- [ ] 1万円/回で予約フォームから予約できる
- [ ] AI が事前鑑定書を生成し、運営者がそれを元にセッションを実施
- [ ] 60分の Zoom セッションが提供される

### F19 — リピート割引 / 紹介プログラム
- [ ] 既存ユーザー向けの2回目以降10〜20%割引がある
- [ ] 紹介リンク経由の登録で双方に特典が付与される

## SNS 運用方針（顔出しNG 前提）

### 媒体選定と役割
| 媒体 | 役割 | 投稿頻度 |
|---|---|---|
| **X (Twitter)** | メイン拡散・短文 tips | 1日 3〜5投稿 |
| **Threads** | X クロスポスト + 独自の長文発信 | 1日 1〜2投稿 |
| **Instagram (Feed + Story)** | ビジュアル世界観の発信、画像メイン | 1日 1投稿 + Story |
| **note** | 長文記事で深い世界観・教養を伝える | 月 2〜4本 |
| **TikTok / Reels** | テキスト+音楽のショート動画 | 週 2〜3本 |
| **LINE 公式** | コアファン化・課金導線 | 毎朝 自動配信 |

### コンテンツ3本柱
1. **教育系（40%）**: 四柱推命・星読みの「知識」を解説、信頼構築
2. **体験系（40%）**: 「今日の星詠み」「今週のメッセージ」など毎日のお守り
3. **販売系（20%）**: PDF レポート / LINE 月額 / コミュニティへの誘導

### KPI（最初の3ヶ月）
- X フォロワー: 1,000人
- Instagram フォロワー: 500人
- LINE 友達: 200人
- 有料転換率: LINE 友達の5%（10人）
- 月額売上: 5万円達成
