# hoshiyomi ブランドガイド

> 「自分の星と対話する」体験を、テキストと画像だけで届けるブランド。
> 顔出しなしの代わりに、世界観の細部で勝負する。

## 1. ブランドコア

| 項目 | 内容 |
|---|---|
| ブランド名 | hoshiyomi（星詠み） |
| タグライン（仮） | 今日の星が、あなたに詠む。 |
| トーン | 凛とした神秘、知的、そっと寄り添う、過剰に煽らない |
| 禁忌 | 断定的予言、不安煽動、医療・健康効果の主張 |

## 2. カラーパレット

| 用途 | 名前 | HEX | Tailwind |
|---|---|---|---|
| 主背景 | night | `#0B1026` | `bg-night` |
| 主背景（深） | night-deep | `#070A1A` | `bg-night-deep` |
| 主背景（明） | night-soft | `#161B3D` | `bg-night-soft` |
| アクセント | violet-mystic | `#3D2B6B` | `bg-violet-mystic` |
| アクセント明 | violet-light | `#6F5BB5` | `bg-violet-light` |
| ハイライト | gold | `#D4AF37` | `text-gold` |
| ハイライト明 | gold-soft | `#E8C766` | `text-gold-soft` |
| 文字（主） | moon | `#F5F3FF` | `text-moon` |
| 文字（副） | moon-dim | `#CFC9E6` | `text-moon-dim` |

**配色ルール**
- 背景は night 系で必ず暗い
- 「黄金（gold）」は強調にだけ使う（タイトル・ボタン縁・星粒など）
- 紫はカード背景、ボタン、グラデの中間色

## 3. タイポグラフィ

| 用途 | フォント | ウェイト |
|---|---|---|
| 見出し（和） | Shippori Mincho | 500 / 700 |
| 本文 | Inter | 400 / 500 |
| 数字（生年月日など） | Inter Tabular | 500 |

- 見出しは縦組み風の余白を意識
- 本文は line-height 1.8 でゆったり
- 「星詠み」「お守り」など固有語は鉤括弧「」で囲み気品を出す

## 4. ビジュアル要素

- **モチーフ**: 月相 / 星粒（粒子）/ 細い金線 / 円
- **写真**: 使わない（顔出しNG前提）。代わりに CSS グラデと SVG 粒子で世界観を作る
- **アイコン**: lucide-react などのミニマル線画のみ
- **アニメーション**: 星のまたたき（twinkle）、ゆっくりした浮遊（drift）

## 5. 文章ルール（重要：景表法・薬機法）

| OK | NG |
|---|---|
| 「〜という傾向があります」 | 「絶対に〜になります」 |
| 「〜と言われています」 | 「100%当たります」 |
| 「自己理解のヒントに」 | 「病気が治ります」 |
| 「お守りとして」 | 「健康効果があります」 |

- 鑑定 AI のプロンプトにもこのルールを必ず注入する
- 「占い」より「星詠み」「メッセージ」「お便り」の語を優先

## 6. SNS 投稿テンプレ（顔出しNG）

- 画像 1:1 もしくは 4:5
- 上下に余白を多く取る
- 中央に星詠みメッセージ（80〜120文字）
- 下に「— hoshiyomi」のサイン
- ハッシュタグは2〜4個に絞る（`#星詠み #今日の運勢 #スピリチュアル #自己理解`）
