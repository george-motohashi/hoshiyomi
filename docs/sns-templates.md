# hoshiyomi SNS 投稿テンプレ 30 本

顔出しNG前提。X / Threads / Instagram 共通で使えるテキストベース。
表記ルール: 断定的予言・誇大表現は使わない（CLAUDE.md 執筆ガイドライン準拠）。

## 投稿の使い方

- そのままコピペ → SNS 投稿
- 「[ ]」の部分は当日の月相・季節・曜日に合わせて置換
- ハッシュタグは媒体ごとに調整（X は4個まで、Instagram は20〜30個）
- LP の URL（`https://hoshiyomi.example.com` → 本番URL）はプロフィールに固定 + 一部投稿で URL を本文に
- 1 投稿につき**改行2回**で読みやすく

```
あなたの“素”の部分。

人前で出る性格よりも、

#星詠み #四柱推命 #自己理解
```

### 02. 西洋占星術の太陽・月・上昇
```
よく言う「太陽星座」だけだと、
本当のあなたは見えないかもしれません。

太陽 → 目指す方向
月 → 内側のあなた
上昇（アセンダント）→ 他人から見えるあなた

3つ揃って、ようやく“あなたの星”です。

#星詠み #ホロスコープ #スピリチュアル
```

### 06. 数秘術の「ライフパスナンバー」
```
生年月日の数字を全部足して、
1桁になるまで足し続ける。

それがあなたの「ライフパスナンバー」。

たとえば 1995/7/15 → 1+9+9+5+7+1+5 = 37 → 3+7 = 10 → 1+0 = 1

「1」は始まりと独立の数。
あなたの数字は、いくつでしたか？

#数秘術 #星詠み #自己理解
```
